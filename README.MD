# 🌍 TravelX - Modern Travel Website

TravelX is a fully responsive travel website built using HTML, CSS, and JavaScript.  
It features a modern UI with glassmorphism, animations, and a mobile-friendly design.

---

## 🚀 Features

- ✅ Fully Responsive (Mobile + Tablet + Desktop)
- 🎨 Glassmorphism UI Design
- 🎬 Smooth Scroll Animations (AOS)
- 📱 Mobile Hamburger Navigation
- 🌍 Destination Cards with Hover Effects
- 🔍 Live Search Filter
- 📩 Contact Form UI
- ⭐ Portfolio-ready Design
